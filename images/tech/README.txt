INSTRUCCIONES PARA LAS IMÁGENES DE TECNOLOGÍAS
===============================================

Coloca aquí los logos de las tecnologías con los siguientes nombres:

1. cypress.png          - Logo de Cypress
2. playwright.png       - Logo de Playwright
3. selenium.png         - Logo de Selenium
4. javascript.png       - Logo de JavaScript
5. python.png           - Logo de Python
6. docker.png           - Logo de Docker
7. github-actions.png   - Logo de GitHub Actions

RECOMENDACIONES:
- Formato: PNG con fondo transparente (preferible)
- Tamaño recomendado: 200x200px o superior
- Las imágenes se redimensionarán automáticamente a 100x100px
- Si las imágenes tienen fondo, preferir fondos oscuros o transparentes
- El contenedor ya tiene un fondo glassmorphic violeta que hace buen contraste

EFECTO VISUAL:
Cada imagen tendrá:
✓ Fondo glassmorphic oscuro con borde violeta
✓ Efecto hover con brillo violeta
✓ Animación de escala al pasar el mouse
✓ Sombra con tono violeta
